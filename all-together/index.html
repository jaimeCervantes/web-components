Usando custom elements, html imports, shadow DOM y tempate

Ejercicio

Crear un componente web, que use template, shadow DOM y tenga estilos para mostrar un Saludo de color Rojo y un comentario extra abajo del saludo

Ejercicio

Crear un componente que reciba como propiedades una imagen(url) y un texto de caption de la imagen de tal manera que el resultado final debe ser la imagen centrado y el texto de caption abajo de la imagen y centrado tambien.

Ejericio

Crear un custom element de un grupo de botones, los cuales sean aceptar, cancelar y cerrar.

  1. El boton de aceptar debe mandar un CustomEvent llamado "onAccept"
  2. El boton de cancelar debe mandar un CustomEvent llamado "onCancel"
  3. El boton de cerrar debe mandar un CustomEvent llamado "onClose"

Esos eventos deben ser escuchados desde la pagina principal

Si es necesario, ocupa shadow DOM para agregar el grupo de botones
